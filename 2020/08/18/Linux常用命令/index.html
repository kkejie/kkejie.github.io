    查看系统信息

    1. 查看物理CPU的个数
      cat /proc/cpuinfo | grep "physical id" | sort | uniq | wc -l
    2. 查看物理CPU内核的个数
      cat /proc/cpuinfo | grep "cpu cores" | uniq
    3. 查看所有逻辑CPU的个数
      cat /proc/cpuinfo | grep "processor" | wc -l
    4. 查看每个物理CPU中逻辑CPU的个数
      cat /proc/cpuinfo | grep 'siblings' | uniq
    5. 查询CPU是否启用超线程
      cat /proc/cpuinfo | grep -e "cpu cores" -e "siblings" | sort | uniq

    linux查找符合条件的文件并删除

    1. 找到根目录下所有的以test开头的文件并把查找结果当做参数传给rm -rf命令进行删除： 

      find / -name “test*” |xargs rm -rf 
find / -name “test*” -exec rm -rf &#123;&#125; \; 
rm -rf $(find / -name “test”)

    2. 如果想指定递归深度，可以这样(只会查找三层目录中符合条件的文件并删除掉！)： 

      find / -maxdepth 3 -name “*.mp3” |xargs rm -rf 
find / -maxdepth 3 -name “test*” -exec rm -rf &#123;&#125; \; 
rm -rf $(find / -maxdepth 3 -name “test”)

    命令

    命令 全拼 含义
    pwd print work directory 打印当前目录 显示出当前工作目录的绝对路径
    ps process status 进程状态，类似于windows的任务管理器,常用参数：－auxf
    ps -auxf / 显示进程状态
    df disk free 其功能是显示磁盘可用空间数目信息及空间结点信息。换句话说，就是报告在任何安装的设备或目录中，还剩多少自由的空间。
    du Disk usage 磁盘使用率
    rpm RedHat Package Management 是RedHat的发明之一
    rmdir Remove Directory 删除目录
    rm Remove 删除目录或文件
    cat concatenate 连锁/查看文件
    cat file1file2>>file3 / 把文件1和文件2的内容联合起来放到file3中
    insmod install module 载入模块
    ln -s link -soft 创建一个软链接，相当于创建一个快捷方式
    mkdir Make Directory 创建目录
    touch touch 创建文件
    man Manual 查看命令操作手册
    su Swith user 切换用户
    cd Change directory 切换目录
    ls List files 文件列表
    mkdir Make directory 创建文件夹
    rmdir Remove directory 删除文件夹
    mkfs Make file system 生成系统文件
    fsck File system check 文件系统检查
    mv Move file 移动文件
    rm Remove file 删除文件
    cp Copy file 拷贝文件
    scp scp -r root@ip:/folder/file /folder 远程拷贝文件，把什么拷贝到哪里去
    ln Link files 链接文件
    fg Foreground 将命令进程号码为N的命令进程放到前台执行
    bg Background 将命令进程号码为N的命令进程放到后台执行
    chown Change owner 更改某个文件或目录的所有者
    chgrp Change group 更改某个文件或目录的用户组
    chmod Change mode 设置权限
    umount Unmount 取消挂载
    tail tail -f 实时读取文件
